# TodoApp

React、TypeScript、Tailwind CSS を使用し、ローカルストレージでデータを永続化した「Todoアプリ」です。
課題提出などを利用者に徹底させるために
ピン機能や，前回アクセスから次のアクセスの間で日付を
跨いでいたときに，課題の追加を促すウィンドウや，
しっかり時間を表示させるところ，また期限までの近さによってソートして表示させています．
